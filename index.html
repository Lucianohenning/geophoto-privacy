Política de Privacidade

GeoPhoto Report

1. Introdução

O GeoPhoto Report é um aplicativo corporativo desenvolvido para registro fotográfico com geolocalização. Respeitamos sua privacidade e estamos comprometidos com a proteção dos seus dados pessoais.

2. Informações Coletadas

2.1 Dados pessoais

Nome completo, e-mail, foto de perfil, cargo (role), login e senha criptografada. Esses dados são enviados e armazenados em um servidor seguro e utilizados apenas para autenticação e controle de acesso. Em caso de desligamento, o acesso é automaticamente inativado.

2.2 Dados de uso e dispositivo

Coletamos informações anônimas sobre desempenho e uso do aplicativo para aprimoramento contínuo.

2.3 Dados de geolocalização

O acesso ao GPS é utilizado exclusivamente para registrar coordenadas nas fotos. Nenhum dado é compartilhado com terceiros.

2.4 Acesso à câmera e galeria

O GeoPhoto Report solicita permissão para usar a câmera e a galeria apenas para capturar ou selecionar imagens. Nenhuma imagem é enviada automaticamente para servidores externos sem ação explícita do usuário.

3. Finalidade do uso dos dados

Os dados são usados para autenticação, controle de acesso, geração de relatórios e melhoria de desempenho.

4. Armazenamento e segurança

Os dados pessoais são armazenados em servidores criptografados. As fotos e relatórios permanecem armazenados apenas no dispositivo do usuário.

5. Compartilhamento de dados

O GeoPhoto Report não compartilha, vende ou transfere informações pessoais a terceiros. Os dados são usados exclusivamente para fins corporativos internos.

6. Permissões e integrações externas

Utilizamos permissões de câmera, galeria, localização e internet, integradas às APIs seguras do Google e Apple, sempre via HTTPS.

7. Direitos do usuário

O usuário pode solicitar acesso, correção ou exclusão de seus dados, além de revogar permissões ou pedir a desativação do cadastro junto à empresa.

8. Retenção e exclusão

Dados no servidor são mantidos enquanto o cadastro estiver ativo. Em caso de desligamento, os dados podem ser mantidos apenas para fins de auditoria interna conforme a LGPD.

9. Contato

Para dúvidas ou solicitações, entre em contato: privacidade@geophoto.app

10. Alterações

Esta Política pode ser atualizada periodicamente. A versão mais recente estará sempre disponível no aplicativo, na seção “Privacidade”.

11. Conformidade Legal

O GeoPhoto Report segue integralmente a Lei Geral de Proteção de Dados (LGPD – Lei nº 13.709/2018) e as diretrizes da Google Play e Apple App Store.
